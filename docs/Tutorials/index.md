# Tutorials

Welcome to the FluidLoom Tutorials. These guides are designed to take you from a beginner to an advanced user of the engine.

## Learning Path

### [1. Getting Started](Tutorial_1_Getting_Started.md)
**Level**: Beginner  
**Time**: 15 mins  
**Goal**: Build the engine and run your first simulation.

### [2. Building a Feature](Tutorial_2_Building_a_Feature.md)
**Level**: Intermediate  
**Time**: 30 mins  
**Goal**: Implement a custom diffusion stencil operation using Lua and GLSL.

### [3. Advanced Usage](Tutorial_3_Advanced_Usage.md)
**Level**: Advanced  
**Time**: 45 mins  
**Goal**: Create a complex Advection-Diffusion simulation with multiple fields.

### [4. Integration Example](Tutorial_4_Integration_Example.md)
**Level**: Expert  
**Time**: 20 mins  
**Goal**: Integrate the engine with external tools by exporting data to OpenVDB files.
